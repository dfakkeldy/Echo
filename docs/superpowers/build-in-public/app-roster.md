# Cross-App App Roster

**Last updated:** 2026-06-27
**Review cadence:** Monthly batching session, then weekly during launch windows

This roster is the single place to decide which apps deserve public attention in
a given month. If an app is not ready for a claim, the calendar should not force
one.

## Status Legend

| Status | Meaning |
|---|---|
| Active | Worth including in the current month. |
| Hold | Keep notes, but do not post unless something material changes. |
| Quiet | No public content this month. |

## Monthly Readiness Questions

- What did this app actually ship since the last calendar refresh?
- Is there a screenshot or clip that proves the claim?
- Is the audience clear enough to know where the post belongs?
- Does the app need recruiting, validation, testers, or just a public record?
- Is this a week to post, or a week to quietly build?

## Default App Priority

1. Echo, because it has the richest weekly proof and existing beta context.
2. Routey, because it has a clear niche audience but needs privacy-safe proof.
3. MacroMark, NS Marks The Spot, and Turn Timer when each has a fresh screenshot or clip.
