# WS2: CI Infrastructure — Implementation Plan

**Goal:** Create a continuous integration (CI) pipeline using GitHub Actions to automatically compile all Xcode schemes and execute the unit and integration test suites on every pull request and push to the `main` branch. This guarantees that future changes do not introduce regressions on iOS, watchOS, or macOS.

---

## Proposed Changes

### GitHub Actions Configuration

#### [NEW] [ci.yml](file:///Users/dfakkeldy/Developer/Echo/.github/workflows/ci.yml)
Create the workflow file under `.github/workflows/ci.yml` with the following configuration:

```yaml
name: CI

on:
  push:
    branches: [ main ]
  pull_request:
    branches: [ main ]

jobs:
  build-and-test:
    runs-on: macos-15
    steps:
      - name: Checkout Code
        uses: actions/checkout@v4

      - name: Select Xcode
        run: sudo xcode-select -s /Applications/Xcode.app

      - name: Cache Swift Package Manager Dependencies
        uses: actions/cache@v4
        with:
          path: ~/Library/Caches/org.swift.swiftpm
          key: ${{ runner.os }}-spm-${{ hashFiles('**/Package.resolved', 'Echo.xcodeproj/project.pbxproj') }}
          restore-keys: |
            ${{ runner.os }}-spm-

      - name: Run iOS Tests
        run: |
          xcodebuild test \
            -project Echo.xcodeproj \
            -scheme "Echo" \
            -destination "platform=iOS Simulator,name=iPhone 16 Pro" \
            -parallelizeBuildables YES \
            -quiet

      - name: Run watchOS Tests
        run: |
          xcodebuild test \
            -project Echo.xcodeproj \
            -scheme "Echo Watch App" \
            -destination "platform=watchOS Simulator,name=Apple Watch Series 10" \
            -parallelizeBuildables YES \
            -quiet

      - name: Build macOS Application
        run: |
          xcodebuild build \
            -project Echo.xcodeproj \
            -scheme "Echo macOS" \
            -destination "platform=macOS" \
            -parallelizeBuildables YES \
            -quiet
```
